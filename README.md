# auth | Vue 3 + TS + Firebase + pinia + tailwind css

Simple auntification, available at:
1. mail + password
2. google

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
